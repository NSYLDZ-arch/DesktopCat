DesktopCat 🐱
A cute, interactive desktop pet that lives on your screen! It walks, runs, eats, and occasionally gets a little dizzy.

Features
Interactive: Feed the cat treats (🐟) and watch it hunt!

Autonomous: The cat wanders around your desktop, takes naps, and meows.

Playful: You can drag the cat around, pet it, and even force it to perform specific actions.

Cross-Platform: Works on both Linux and Windows.

Installation & Running
For Windows users

Go to the Releases page

Download the exe file (and other png file)

put the png files in an animtions folder

Double-click DesktopCat.exe to run.

For Linux (Lubuntu) Users
Ensure you have Python 3 installed.

Install the required dependencies:

Bash
pip install PyQt5 Pillow
Run the script:

Bash
python3 pet.py
Note: Ensure the animations/ folder is in the same directory as the script.

How to use
Drag: Click and hold the left mouse button to pick up your cat. If you shake it too fast, it will get dizzy!

Feed: Right-click the cat and select "Feed Cat" to drop a fish treat. The cat will hunt it down!

Interact: You can use the right-click context menu to manually change the cat's actions or force it to stalk your mouse cursor.

For Windows Users!

Just go in the dist folder and run the DesktopCat file.
